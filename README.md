
EVAN'S FUNNEST AWESOME CRAFT!!!
=======

A Minecraft overhaul mod, modpack, & resource pack about Evan and His Funnest Awesome Craft. 
Currently in the process of being ported from Fabric 1.20.2 to NeoForge 1.21.1.
When completed, it will add a custom magic system built around clumbing, a mouth & tooth system that allows the player to dynamically change their stats, several custom dimensions, biomes, and structures, and hopefully more.
It will also change every vanilla texture and sound file to make the game look both cute and cool at the same time, and will overhaul a number of the basic mechanics like movement, sleeping, or enchanting.
